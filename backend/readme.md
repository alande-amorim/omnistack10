Backend DevRadar
===

[< back](../)

## Installation
```
$ npm install
```

## Configuration
Use .env.example as a template for your configs and replace as you wish:
```
$ mv .env.example
$ vim .env
```

## Running
```
$ yarn dev
# or
$ npm run dev
```


## Exposed APIs:
``` 
get /devs
```
```
post /devs
```
```
get /search
```
