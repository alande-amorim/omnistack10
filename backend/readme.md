# DevRadar
Omnistack #10 week Project

## Backend

### Installation
After cloning, install it running:
```
$ cd backend
$ npm install
```

### Configuration
Make a copy of .env.example and add your configurations:
```
$ mv .env.example
$ vim .env
```

### Running
```
$ yarn dev
# or
$ npm run dev
```


### Exposed APIs:
``` 
get /devs
```
```
post /devs
```
```
get /search
```
